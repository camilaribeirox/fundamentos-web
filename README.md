# 💻 Desenvolvimento Web

Repositório destinado a guardar os projetos e atividades práticas da disciplina de Desenvolvimento Web do curso de Análise e Desenvolvimento de Sistemas.

## 📚 Sobre o repositório
Aqui registro minha evolução prática na disciplina, documentando os exercícios feitos em aula. Por enquanto, o foco principal dos estudos tem sido a estruturação de páginas utilizando **HTML**, além de conceitos iniciais de estilização.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3
